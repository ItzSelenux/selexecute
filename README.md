# selexecute
selexecute is a bash script that execute some commands according to you ~/.config/selexecute.conf
Options: 
    editor
    -e  Open text editor
    files
    -f  Open file manager
    launcher
    -l  Open applications launcher
    music
    -m  Open applications menu
    screenshot
    -s  Open screenshooter
    terminal
    -t  Open a terminal emulator
    web
    -w  Open web browser
